# SugarHub
The Sugarcane Disease Detection System is a web-based machine learning project designed to identify diseases in sugarcane leaves using image processing and deep learning techniques. The system allows users to upload leaf images through a simple web interface, where a trained CNN (Convolutional Neural Network) model analyzes the image and predicts the type of disease with high accuracy.

The project is built using Flask for the backend, HTML, CSS, and JavaScript for the frontend, and TensorFlow/Keras for the deep learning model. It helps farmers and agricultural experts quickly detect plant diseases, enabling early treatment and improving crop yield.

The application is deployed as a web platform, where users can easily access the system through a secure HTTPS link after deployment.

Frontend: HTML, CSS, JavaScript
Backend: Flask (Python)
ML Model: TensorFlow / Keras (CNN)
Database: MySQL 
The application is running on a local development server.
It is accessible through the URL: http://127.0.0.1:5000/
This URL represents the homepage of the SweetCaneHub system
